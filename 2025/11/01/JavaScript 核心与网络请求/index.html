一、ES6 核心特性

1.1 变量声明：let 与 const

ES5 的 var 存在的问题：

  1. 变量提升（Hoisting）
  2. 无块级作用域
  3. 可重复声明

let 的特性：

  • ✅ 块级作用域：仅在 {} 内有效
  • ✅ 不可重复声明
  • ✅ 无变量提升（存在”暂时性死区”TDZ）
  • ✅ 不影响全局对象

const 的特性：

  • ✅ 具备 let 的所有特性
  • ✅ 必须初始化
  • ✅ 不可重新赋值

⚠️ 重要澄清：const 保证的是变量指向的内存地址不变，不是值不可变！

四、AJAX 基础

4.1 什么是 AJAX？

AJAX：异步的 JavaScript 和 XML（Asynchronous JavaScript And XML）

核心特性： 可以在不重新刷新页面的情况下与服务器通信，进行数据交换或页面更新。

4.2 传统 Web 请求 vs AJAX 请求

特性 传统 Web（同步） AJAX（异步）
发送方式 表单或超链接 XMLHttpRequest 对象
页面刷新 会重新载入 无需重新载入
响应内容 完整页面 只响应需要的数据
用户体验 需等待响应 可继续操作

4.3 XMLHttpRequest 使用

基本流程：

请求状态码 (readyState)：

状态码 描述

1 已打开（open 已调用）
2 已发送（send 已调用）
3 响应中
4 完成
