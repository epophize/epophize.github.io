一、ECharts 简介

1.1 什么是 ECharts？

ECharts（Enterprise Charts）是由百度开源、现由 Apache 基金会维护的一款基于 JavaScript 的数据可视化库。

特点：

  • ✅ 开源免费
  • ✅ 支持多种图表类型
  • ✅ 高度可定制
  • ✅ 响应式 & 交互性强（支持缩放、拖拽、提示框等）

官方网站： https://echarts.apache.org/zh/index.html

1.2 图形库对比

类型 库 技术
2D 图形 ECharts, HCharts SVG, Canvas
3D 图形 Three.js, Cesium.js Canvas + WebGL
