一、Node.js 基础

1.1 什么是 Node.js？

Node.js 是一个 JavaScript 运行环境 (runtime)，让 JavaScript 可以开发后端程序。

  • 基于 Google V8 引擎
  • 事件驱动、非阻塞 I/O 模型
  • 轻量、高效

1.2 Node.js 的作用

优势 说明
前端必备技能 构建工程化开发环境，具备后端开发能力
学习成本低 语法完全是 JS 语法
功能丰富 动态网站、API 接口、云计算、游戏开发、区块链、即时通讯等

核心功能：

  • 创建 Web 服务器
  • 定义路由
  • 托管静态资源
