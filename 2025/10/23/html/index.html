📘 HTML 核心知识点整理

一、HTML 基础概念

概念 说明
全称 HyperText Markup Language（超文本标记语言）
定位 用于创建网页的标准标记语言，不是编程语言
运行环境 运行在浏览器中，默认由浏览器解析并渲染成可视页面
组成 由一套标记标签（Tags）和文本内容组成，用于描述和编辑网页结构
文档别称 HTML 文档通常被称为 Web 页面

二、浏览器与内核

  • 常见浏览器：Chrome、Firefox、Safari、IE、Edge、360、QQ、UC、搜狗、Opera 等。
  • 浏览器核心引擎：
    • 🖥️ 渲染引擎（排版引擎）：获取 HTML/CSS 内容，解析并显示网页布局与样式。
      • Chrome / 新版 Edge → Blink
      • Firefox → Gecko
      • IE → Trident
      • Safari → WebKit
    • ⚙️ JS 引擎：负责解析和执行 JavaScript 代码。
    • 🖥️ w3c:万维网联盟,W3C是对web进行标准化的国际组织

三、HTML 标签与元素

  • HTML 标签（Tag）：通常由尖括号 < > 包裹关键词组成，如 <html>。
  • 成对结构：大多数标签成对出现
    • <标签名> → 开始标签（开放标签）
    • </标签名> → 结束标签（闭合标签）
    • 完整结构：<开始标签>文本内容</结束标签>
  • HTML 元素（Element）：通常与“标签”混用。严格来说，元素 = 开始标签 + 内容 + 结束标签。
  • 单标签：部分标签无需闭合，如 <br>、<hr>、<img>、<input>。

四、标签分类与嵌套规则

1. 显示特性分类

类型 特性 常见标签
行内元素（Inline） 一行可排列多个，不自动换行，默认不支持设置宽高 span, strong, b, em, i, var, u, ins, del, s, sup, sub, big, small, a, img, label, input, select, button, textarea
块级元素（Block） 独占一行，自动换行，支持设置宽高 h1~h6, p, hr, div, address, pre, table, tr, caption, thead, tbody, tfoot, form, option

💡 注：现代 CSS 中还有 inline-block（行内块）等类型，此处按传统分类整理。

2. 嵌套规则

  1. 行内元素可嵌套行内元素，不可嵌套块级元素。
  2. 块级元素可嵌套块级元素，也可嵌套行内元素。
  3. 特殊限制：h1~h6、p、address 只能嵌套行内元素，不可嵌套块级元素。
  4. 块级元素嵌套时，同级元素类型应保持一致（块配块，行配行）。

五、常用基础标签

1. 标题与段落

2. 换行与分割

  • <br>：强制换行（单标签）
  • <hr size="30">：水平分割线（size 控制高度，HTML5中建议用CSS替代）

3. 文本格式化

效果 语义化标签（推荐） 纯样式标签
加粗 <strong> <b>
倾斜/强调 <em> <i>, <var>
下划线 <ins>（插入） <u>
删除线 <del>（删除） <s>, <strike>
字号 <small>（小）, <big>（大） -
上下标 <sub>（下标）, <sup>（上标） -
预格式 <pre>（保留空格与换行） -
地址 <address> -
内联容器 <span>（纯文本/样式钩子） -
块级容器 <div>（布局/盒模型） -

六、图片与超链接

🖼️ 图片标签 <img>（单标签）

属性 说明
src 图片路径/URL（绝对或相对路径）
width / height 宽高（默认单位 px）。仅设其一，另一项会等比例缩放
alt 图片加载失败时的替代文本（辅助功能 & SEO必备）
title 鼠标悬停时显示的提示文本
draggable 是否允许拖拽，值为 true 或 false

📁 路径说明：

  • ./ 当前目录，../ 上一级目录，/ 根目录或下一级
  • 绝对路径：完整服务器地址（如 https://example.com/img.png）

🔗 超链接标签 <a>

属性 说明
href 跳转目标地址（绝对/相对路径）。仅引用，点击时才加载
target 打开位置：_self（默认当前页）、_blank（新窗口）、_top（顶级窗口）、_parent（父窗口）
download 触发文件下载
media 规定链接资源的媒体类型

🔖 锚点定位

  1. 定义锚点：在目标元素添加 id="锚点名"（或旧版 <a name="锚点名">）

    ⚠️ id 命名规范：以字母或 _ 开头，不可为纯数字或中文。如 one, _section1, menu_2

  2. 跳转链接：<a href="#锚点名">跳转到指定位置</a>

🎨 默认样式：未访问蓝色带下划线 → 访问中红色 → 访问后紫色

七、列表标签

类型 标签 常用属性
有序列表 <ol> + <li> type（1, A, a, I, i）
reversed（倒序）
start（起始数字）
无序列表 <ul> + <li> type（disc实心圆, circle空心圆, square实心方块）
列表项 <li> type（覆盖父级样式）, value（自定义序号值）

八、表格标签

基础结构

标签 说明 常用属性
<table> 表格容器 border, bgcolor, width, height, align, cellpadding（内容与边框距离）, cellspacing（单元格间距）
<tr> 表格行 bgcolor, align（水平）, valign（垂直：top/middle/bottom）
<th> 表头单元格 继承 <td> 所有属性，默认文本居中加粗
<td> 数据单元格 bgcolor, align, valign, width, height
⚠️ 设置单个 td 宽高会影响整列/整行
<caption> 表格标题 align（top/bottom）
<thead> <tbody> <tfoot> 语义化分区 增强可读性，利于样式控制与打印分页

🔗 合并单元格

  • colspan="n"：向右合并（跨列）
  • rowspan="n"：向下合并（跨行）

    💡 标准文档流：元素默认从左到右、从上到下自动流式排列。

九、表单标签（收集用户输入）

📦 表单容器 <form>

属性 说明
action 数据提交的服务器地址/路径
method 提交方式：get（参数附在URL） / post（参数在请求体，更安全）

📝 常用表单元素

1. <input> 单行输入框

type 值 说明
text 普通文本
password 密码（字符隐藏）
checkbox 多选框（配合 name 分组，可用 label 绑定文字）
radio 单选框（必须同名 name 实现互斥）
file 文件上传
number 数字输入
date / datetime-local 日期 / 日期时间
image 图片提交按钮
submit / reset / button 提交 / 重置 / 普通按钮

通用属性：

  • max/min/step（适用于 number/range）
  • maxlength：最大字符长度
  • placeholder：输入提示文本
  • value：默认值/预输入值
  • checked：默认选中（仅 checkbox/radio）
  • disabled：禁用（不可交互，值不提交）
  • readonly：只读（可交互，值正常提交）
  • name：字段名称（提交时作为键名）
  • autofocus：自动获取焦点
  • required：必填校验
  • size：输入框可见字符宽度（默认20）

2. 其他表单控件

标签 说明 核心属性
<select> 下拉列表 内部嵌套 <option> 或 <optgroup>
<option> 下拉选项 selected（默认选中）
<optgroup> 选项分组 label（分组标题）
<textarea> 多行文本域 rows（行高）, cols（列宽）
支持 autofocus, placeholder, readonly, required, disabled, maxlength
<label> 关联文本 for="对应元素id" 或包裹元素，扩大点击范围
<button> 按钮 默认类型 submit，可用 type="button" 改为普通按钮
<fieldset> 表单分组 自带外边框
<legend> 分组标题 配合 <fieldset> 使用

十、头部文档与内联框架

📄 头部标签（位于 <head> 内，不直接显示在页面）

标签 作用
<title> 文档标题：显示在浏览器标签页、收藏夹、搜索引擎结果页。HTML文档必需。
<meta> 元数据：描述网页字符集、关键词、作者、视口设置等。不渲染在页面。
<link> 引入外部资源。常用：<link rel="shortcut icon" href="path/to/icon.ico">（网站图标，常用尺寸 16×16 或 32×32）

🖼️ 内联框架 <iframe>

属性 说明
src 嵌入网页的路径（绝对/相对）
width / height 框架宽高（默认 px）
frameborder 是否显示边框：1（默认有边框） / 0（无边框）
name 框架名称，可被 <a target="name"> 或 <form target="name"> 定向加载
