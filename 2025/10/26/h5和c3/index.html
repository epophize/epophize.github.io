HTML5 & CSS3 部分

🟦 HTML5 核心知识点

1. 语义化标签

标签 说明 默认显示类型
<header> 页面或区块的头部 block
<footer> 页面或区块的底部 block
<nav> 导航链接区域 block
<section> 文档中的通用独立区块 block
<article> 可独立分发或复用的内容（如博客、新闻） block
<aside> 侧边栏或附属内容 block
<main> 页面主体内容（每个页面仅一个） block
<figure> & <figcaption> 图文组合及说明 block
<details> & <summary> 可折叠交互内容 block
<mark> 高亮文本 inline
<time> 日期/时间机器可读 inline

📝 修订提示：原内容中 aricle 已修正为 article。语义化核心作用：提升 SEO、增强可访问性（A11y）、优化代码可读性与维护性。

2. 多媒体标签

标签 核心属性 说明
<img> src, alt, width, height, loading="lazy" 图片标签，必须写 alt 用于无障碍与 SEO
<video> src, controls, autoplay, loop, muted, preload, poster 视频标签。现代浏览器要求 autoplay 必须搭配 muted
<audio> src, controls, autoplay, loop, preload 音频标签
<source> src, type, media 多格式资源回退，嵌套于 video/audio/picture 内

📝 修订提示：

  • 原描述“行内块级元素”不准确。img/video/audio 属于替换元素（Replaced Elements），默认 display: inline，但具有固有宽高，实际开发中常设为 display: block 或 inline-block。
  • <source> 的 type 应使用标准 MIME 类型（如 video/mp4, audio/webm），配合 media 可实现响应式媒体加载。

3. 表单结构标签 <form>

属性 说明
action 表单提交的 URL 地址
method GET（URL 传参，适合查询） / POST（请求体传参，适合提交/文件）
enctype application/x-www-form-urlencoded（默认） / multipart/form-data（含文件上传） / text/plain
target _self（默认） / _blank / _parent / _top
autocomplete on / off 控制浏览器自动填充
novalidate 提交时关闭浏览器默认表单校验

📝 补充：form 默认 display: block。现代开发常配合 FormData API 实现异步提交。

4. 表单元素

元素 常用 type / 属性 说明
<input> text, password, email, tel, url, number, date, datetime-local, time, month, week, color, range, search, file, checkbox, radio, submit, reset, button, hidden 单行输入/控件
<textarea> rows, cols, wrap, maxlength, placeholder, resize 多行文本域
<select> & <option> multiple, size, value, selected, disabled 下拉选择
<datalist> id 配合 <input list="id"> HTML5 新增，提供输入建议下拉
<label> for="id" 或包裹 <input> 提升可访问性与点击热区
<fieldset> & <legend> - 表单分组与标题

📝 修订提示：

  • 原内容 datatime-local → datetime-local，seach → search。
  • input, textarea, select 默认显示类型多为 inline 或 inline-block（浏览器实现略有差异），实际开发中常用 CSS 统一重置为 display: block 或 inline-block 便于布局。

5. 表单常用属性汇总

属性 作用 适用元素
name 提交时的键名（Key） 所有可提交控件
value 控件的初始值/提交值 input, option, textarea, button
placeholder 输入提示文本 input, textarea
required 必填校验 大多数输入控件
checked 默认选中 radio, checkbox
selected 默认选中 option
disabled 禁用，不参与提交 所有表单控件
readonly 只读，值可提交 input, textarea
min / max 数值/日期范围限制 number, date, range
step 步进值 number, range, date 系列
minlength / maxlength 字符长度限制 input[type=text/email/tel], textarea
pattern 正则表达式校验 input（文本类）
autofocus 页面加载自动聚焦 单个元素
form 关联外部 <form> 的 id 表单控件

🟪 CSS3 核心知识点

1. 渐变（Gradients）

📝 补充说明：

  • 方向支持关键字：to top, to bottom, to left, to right, to top left 等，或角度 45deg。
  • 颜色可带位置百分比，未指定则自动均分。
  • 可配合 background-size 实现条纹、网格等背景效果。

  • 居中方案中 Y 轴也需偏移，应为 translate(-50%, -50%)。
  • transform 不影响文档流，仅生成视觉层，适合动画与布局微调。

3. 过渡（Transition）

参数 说明
property 过渡的属性（all, opacity, transform, background-color 等）
duration 动画时长（0.3s, 300ms）
timing-function 速度曲线：ease（默认）, linear, ease-in, ease-out, ease-in-out, cubic-bezier()
delay 延迟时间（可选）

📝 注意事项：

  • 过渡必须绑定在触发状态（如 :hover, :focus, .active）或 JS 切换 class 时生效。
  • 建议明确指定属性而非全用 all，提升性能。
  • transform 和 opacity 触发 GPU 加速，性能最优。

📦 重要补充（现代开发高频特性）

HTML5 补充

特性 说明
<picture> & <source media> 响应式图片加载（按屏幕/分辨率切换）
loading="lazy" 图片/iframe 懒加载（原生支持）
<dialog> 原生模态对话框，支持 .showModal()
Web Storage localStorage / sessionStorage 替代部分 Cookie 场景
FormData & fetch 现代表单异步提交标准方案

CSS3 补充

特性 说明
@keyframes & animation 关键帧动画（animation: name duration timing-function delay iteration-count direction fill-mode play-state）
Flexbox / Grid 现代布局核心，替代传统 float/position 方案
box-shadow / text-shadow 阴影效果，支持多层
calc() 动态计算（如 width: calc(100% - 2rem)）
var() 自定义属性 CSS 变量，支持主题切换与响应式
@media 媒体查询 响应式设计基石
filter / backdrop-filter 滤镜与毛玻璃效果
object-fit / object-position 替换元素内容适配（替代传统背景图 hack）
